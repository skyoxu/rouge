# Rouge v0 参考：最小内容集清单（非 SSoT，不进门禁）

- 本清单仅用于保证“有东西可跑、可测、可复现”，不作为设计承诺。
- 若与 PRD/Overlays/Tasks/Contracts 不一致：以 SSoT 为准。

## 1. v0 最小内容集（覆盖当前任务语义）

### 1.1 卡牌（Cards）
- 至少 8–12 张基础牌：
  - 伤害牌 4–6
  - 防御/护盾牌 2–4
  - 功能牌 2（抽牌/上状态/能量）

### 1.2 敌人（Enemies）
- 至少 4 个普通敌人（2 个低难度、2 个中难度）
- 至少 1 个 Boss（章节结算）
- 每个敌人至少 2 种可展示的意图（Attack/Defend/Buff/Debuff 任意 2 种）

### 1.3 遭遇（Encounters）
- 至少 3 个遭遇：单怪 / 双怪 / Boss

### 1.4 地图与节点（Map/Nodes）
- 至少 1 个 Act（v0 可先单章）
- 节点类型至少覆盖：Battle / Event / Shop / Rest / Boss
- 每个非终点节点必须能确定下一步（`nextNodeIds` 非空）

### 1.5 奖励与商店（Rewards/Shop）
- 至少 1 个奖励池：金币 + 选卡
- 至少 1 个商店目录：3–5 个条目（卡/治疗/移除卡等）

### 1.6 事件抉择（Events/Choices）
- 至少 2 个事件：每个事件至少 2 个选项
- 至少 1 个选项包含“代价 + 收益”组合（例如失去 HP 获得金币）

### 1.7 跑局存档（Save/Load）
- 至少 1 个可序列化的跑局快照样例（用于 Save/Load 与版本迁移测试）
